###  Electronics-Clothing-Sales-Dashboard

# 📊 Electronics & Clothing Sales Dashboard

## 📌 Project Overview

This project demonstrates an end-to-end Data Analytics workflow by analyzing **Electronics and Clothing Sales** data. The project covers data storage, SQL analysis, data cleaning, visualization, and dashboard publishing using industry-standard tools.

---

## 🎯 Objectives

- Analyze sales performance across different product categories.
- Identify top-performing products, cities, and states.
- Monitor sales, profit, and quantity sold.
- Create an interactive dashboard for business decision-making.
- Publish the dashboard to Power BI Service for online access.

Data preprocessing was performed using **Pandas** and **NumPy**.

Cleaning tasks include:

- Handling missing values

- Removing unnecessary spaces
